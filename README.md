# photo-cleanup

Use `photo-cleanup` for:

- Organize files through Model, Date and Location.
- Compare parts of data to find matches and diffs.

## Installation

Install with npm:

``` bash
$ npm i photo-cleanup -g
```

## Usage

``` bash
# Organize files
$ photo-cleanup <fold> <?destFold>
```

``` bash
# Compare parts
$ photo-cleanup compare <fold1> <?fold2>
```
